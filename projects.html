Hangman Game Project

As part of **CSI 2300: Software Development**, I teamed up with partners to create a **Hangman game with a graphical user interface (GUI)** using **Java** and **JavaFX**. Our goal was to make the game interactive and user-friendly, and we implemented several features to enhance the experience:

  • Multiple **difficulty levels** (easy, medium, hard) to challenge players of different skill levels.
  • Various **categories** of words, including movies, animals, and technology, so the game stays fun and varied.
  • **Hint system** to help players when they get stuck, adding a strategic element.
  • Clean, intuitive **GUI design** with buttons, text fields, and visual feedback for correct/incorrect guesses.
  • Collaborative workflow using GitHub to **manage code, track changes, and merge contributions** from each team member.

This project taught me a lot about **team communication**, **problem-solving**, and designing software that balances functionality and user experience. I focused on implementing the **GUI and hint system**, while coordinating with teammates on game logic and category setup.
